# Interactive Survival Explorer

A Shiny application demonstrating survival analysis using the R survival package.

## Features
- Kaplan-Meier survival estimation
- Stratified survival curves
- Risk table visualization
- Built-in lung cancer dataset

## Packages Used
- survival
- survminer
- shiny
- dplyr

## Future Extensions
- Cox proportional hazards modeling
- User-uploaded datasets
- Interactive hazard ratio interpretation
- Model diagnostics
